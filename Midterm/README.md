# Tic Tac Toe AI(Minimax) Game

This project is python based Tic Tac Toe game where you play against AI which uses minimax algorithm.

## Libraries

You need to install pygame and numpy libraries to play the game.

## How to play

* You are player O and AI is X
* Run the code file and click on a square at the opening screen.
* AI will make its move and so on.
* When the game end close the screen or press 'r' to restart the game.
